Creating the Angular testApp :
------------------------------
- Created the base folder/file structure :
    - app folder to hod all custom scripts.
    - resouces flder to hold all resources
        - styles
        - images
        - scripts
    - index.html file
    
- Initialize the npm modules for the project :
    - Open the folder in the command propmt and type in 
        npm init
    - Fill in the name, version and other details and thi results in the creation of the config file : package.json
    - Add the required libraries :
        npm install jquery -S
        npm install angular -S
        npm install angular-ui-router -S
    - Add the scripts to the index.html file.
